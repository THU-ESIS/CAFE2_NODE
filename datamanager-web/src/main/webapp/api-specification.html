按条件筛选

路径

GET /api/v1/modelfile/query

参数
page
  • 作用：指定当前筛选结果的页码
  • 格式：page=0
  • 字段说明：必选；起始值为0
pageSize
  • 作用：指定当前筛选结果的每一页的行数
  • 格式：pageSize=20
  • 字段说明：必选
institute
  • 作用：指定一组institute作为过滤条件
  • 格式：institute=a&institute=b
  • 字段说明：可选、可多选
model
  • 作用：指定一组model作为过滤条件
  • 格式：model=a&model=b
  • 字段说明：可选、可多选
experiment
  • 作用：指定一个experiment作为过滤条件
  • 格式：experiment=a
  • 字段说明：可选、不可多选
frequency
  • 作用：指定一个frequency作为过滤条件
  • 格式：frequency=a
  • 字段说明：可选、不可多选
modelingRealm
  • 作用：指定一个modelingRealm作为过滤条件
  • 格式：modelingRealm=a
  • 字段说明：可选、不可多选
ensembleMember
  • 作用：指定一组ensembleMember作为过滤条件
  • 格式：ensembleMember=a&ensembleMember=b
  • 字段说明：可选、可多选
variableName
  • 作用：指定一组variableName作为过滤条件
  • 格式：variableName=a
  • 字段说明：可选、不可多选
